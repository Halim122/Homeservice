# Home Services Provider System

## Overview
The Home Services Provider System is a web-based platform developed to connect homeowners with service providers. The system allows users to access home services, request bookings, and manage service interactions through a centralized digital platform.

## Problem Statement
Many homeowners face challenges finding reliable service providers and managing service requests manually. This system provides a platform that simplifies the process by connecting clients and service providers while improving service management.

## Features

### Client Features
- User registration and login
- Browse available services
- Book home services
- Manage booking requests
- Track service status

### Service Provider Features
- Provider registration and login
- Manage service information
- Receive and manage booking requests

### Admin Features
- Admin dashboard
- Manage users and service providers
- Manage services
- Monitor booking activities

## Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap

## System Modules
- User Authentication Module
- Service Management Module
- Booking Management Module
- Admin Management Module
- Database Management Module

## Database
The system uses MySQL database to store:
- User information
- Service provider details
- Service records
- Booking information

## Project Purpose
The system was developed to improve access to home services by providing a reliable platform where homeowners can connect with service providers and manage service requests digitally.
